# Covista

A simple React app for visually organizing (US) national and state COVID-19 data.

## Run locally

```bash
git clone git@github.com:kenziebottoms/covista.git
cd covista
npm i
npm start
```

## Run tests

> Assuming you've installed it locally already.

```bash
npm test
```

## Resources
